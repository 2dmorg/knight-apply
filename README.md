本项目为骑士申请移动端前端部分。

### 说明

- **编码风格** ：详见`.editorconfig`配置（2个空格缩进，行末自动去空格（除markdown文件），最后一行自动换行）
- **移动端适配方案** ：使用了淘宝的`flexible可伸缩布局方案`
- **涉及的技术点**：`React`、`ES6`、`Webpack`、`PostCSS`、`Flexible`、`Classnames...`
- **一些供参考的文章** ： [使用Flexible实现手淘H5页面的终端适配][1](大漠)
### 附

- **运行** ：npm start（别忘记装依赖模块）
